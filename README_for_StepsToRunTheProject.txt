1. Pension Management Portal (Front-end in ANGULAR):-
   
   -> Import this in VS Code.
   -> Run "npm install" to install the node modules in the angular project.
   -> Run "ng serve" to lauch the application on localhost 4200.

2. Spring projects:-

   a. Authorization Service	-> (Port 8081)
   b. Process Pension Service	-> (Port 8082)
   c. Pensioner Detail Service	-> (Port 8083)
   
   -> Import these as Maven projects in eclipse and run them on their respective ports.
